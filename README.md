### 简介

通过WIFI无线通信，实现对所有子系统的连接整合，同时实现通信的全面覆盖

---

## 组成

#### NAPT

liblwip_open_napt.a 库文件

#### repeater

用于wifi中继和相关通信控制

#### PEAP2PSK

WPA2-PEAP to WPA2-PSK converter

#### NaCl

NaCl crypto lib for the ESP8266

For documentation of the API see: https://nacl.cr.yp.to/
